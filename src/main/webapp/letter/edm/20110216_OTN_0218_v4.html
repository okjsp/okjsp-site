오라클이 Java 개발자를 위해 준비한
OTN Java Developer Day가 이번주 금요일에 개최됩니다.  

오라클과 썬이 하나가 되어 처음으로 개최하는 이번 행사에서는 지난해 9월과 12월 샌프란시스코와 베이징에서 열린 Java One 과 Oracle DEVELOP 행사에서 소개된 JDK7와 함께 Java의 플랫폼 미래를 제시합니다.

이 번 세션에서는 오라클 본사의 Java 담당자와 국내의 Java 전문가를 초빙하여, Java관련 최신의 내용을 전하여 드리겠습니다. (행사 등록 페이지에서 아젠다를 참고하세요)

또한 발표자와의 만남 등 Java 개발자들을 위한 다양한 부대 이벤트를 준비하였으니, 참석하셔서 오라클이 제안하는 Java의 미래를 공유하시길 바랍니다.

*본 행사는 Java 개발자들을 위한 행사로서, 제한된 좌석수로 인해 행사 참가신청 후 등록확인을 받은
개발자께서 참석하실 수 있습니다.
