인사말
Java 개발자들을 위해 준비했습니다.

OTN Developer Java Workshop에 Java 개발자 여러분을 초대합니다. 

특히 오는 5월 3일~4일, JCP(Java Community Process)에 참석하시는 Patrick Curran및 오라클 본사의 주요 인사를 모시고, 자바의 현재 이슈와 미래에 대한 세션을 준비 하였습니다. 아래 아젠다를 참고하시고 5월 2일 월요일 대한상공회의소에서 열리는 OTN Developer Java Workshop 에서 만나 뵙기를 희망합니다.

•본 행사는 Java 개발자들을 위한 행사입니다.

Speaker 소개
Patrick Curran - Software Development Director
Patrick Curran는 JCP(자바기술표준협회)의 의장으로서 JCP 프로그램, 회원 관리, 특별한 관리자와 전문가 지도, 집행위원회 미팅 안내 등 모든 프로그램 경영을 총괄 합니다.
Patrick은 썬(현 오라클)에서 20년 가까이 일하였으며 총 25년 이상 소프트웨어 산업분야에서 일해 왔습니다. 그는 conformance testing 에서 오래된 기록을 보유하고 있으며 가장 최근에는 썬의 고객 소프트웨어 그룹 안에서 Java Conformance 엔지니어링 팀을 이끌고 있습니다. 그는 또한 JAVA 에 관련된 순응성과 양립가능성에 대한 정책과 전략에 대한 책임이 있는 썬의 Conformance 의회의 의장을 맡고 있습니다.
Patrick은 여러 컨소시엄과 W3C (월드와이드웹 컨소시엄)과 같은 커뮤니티, OASIS 협회 같은 곳에 활동적으로 참가하고 있습니다.
 
Roger Riggs - Principal Engineer
로저 릭스는 오라클 기술팀의 주요 구성원으로서 자바 구조와 임베디드 및 모바일 장치의 사양에 관한 일을 하고 있다. 그는 많은 오라클 JCP 소형장치 사양의 스펙리드(spec lead)이다. 그는 1995년 이후로 자바 시스템을 설계 및 이행하고 있고, 자바 언어, 라이브러리 및 자바 가상 머신에 관한 광범위한 지식을 갖고 있다.
