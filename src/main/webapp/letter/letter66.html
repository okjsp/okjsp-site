6. pjax 를 아시나요?
pushState + ajax = pjax 입니다.
HTML5 의 window.history.pushState() 메소드와 ajax 가 만났습니다.
뒤로가기 버튼을 이용할 수 없고, 검색인덱스를 생성할 수 없는 ajax 의 한계를 극복했고, hashbang 의 URL hack 을 쓰지 않고도, URL 전환을 처리할 수 있습니다.
